Handy Shortcuts

...sort of a walkthrough

A list of handy keyboard shortcuts for the Command Line and Code Editor

Command Line Interface

Try these on the command line

Copy

To copy a line from the terminal
First, highlight the line, then...

Ctrl + Shift + c

Paste

To paste a line to the terminal

Ctrl + Shift + v

Navigate to Home Directory

cd

Move to the home directory from anywhere

Make Directory

mkdir newDirectory

Create a new directory within the current directory

View Directories

dir

View directories and files located within the current directory

Change to Specicfic Directory

cd newDirectory

Navigate to our newly created directory

Leave Directory

cd ..

Navigate to the parent directory

Copy Directory

cp -r newDirectory/ copiedDirectory

Copy the original directory

Rename Directory

mv copiedDirectory/ renamedDirectory

Rename the copied directory

Move Directory

mv renamedDirectory/ newDirectory

Move the renamedDirectory inside the newDirectory

Create File

touch newFile

create a new file in the current directory

These commands should be run from within the project directory

Initialze a new Git Repository

git init

Check Git Status

git status

Add All Unstaged Files to be Committed

git add .

Discard All Changes

git checkout .

Commit Changes

git commit -am 'description of changes'

Push Commit

git push

Pull from Remote Repository

git pull
